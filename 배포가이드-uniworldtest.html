uniworldtest.co 도메인으로 웹사이트를 실제 인터넷에 공개하는 전체 과정

📋 배포 요약

배포 파일: uniworld-deploy.zip (16MB, 이미지 최적화 완료)

호스팅: Netlify (무료 플랜으로 충분)

도메인: uniworldtest.co (별도 구매 필요)

예상 소요 시간: 30분~1시간

Step 1. Netlify에 웹사이트 배포하기

Netlify는 정적 웹사이트를 무료로 호스팅할 수 있는 서비스입니다. 가장 빠르고 쉬운 방법입니다.

1

Netlify 회원가입

https://app.netlify.com/signup 에 접속하여 회원가입합니다.

GitHub, GitLab, 이메일 중 편한 방법으로 가입하세요.

2

ZIP 파일로 즉시 배포 (가장 간단한 방법)

https://app.netlify.com/drop 에 접속합니다.

uniworld-deploy.zip 파일을 화면에 드래그 & 드롭하면 바로 배포됩니다!

💡 핵심: ZIP 파일을 풀지 않고 그대로 드래그하면 됩니다. 업로드 후 약 30초~1분이면 배포 완료됩니다.
3

배포 확인

배포가 완료되면 Netlify가 자동으로 임시 URL을 부여합니다:

# 예시 URL (실제로는 랜덤 이름이 부여됩니다)
https://random-name-12345.netlify.app

이 URL로 접속하여 웹사이트가 정상적으로 표시되는지 확인하세요.

✅ 여기까지 완료하면 웹사이트가 이미 인터넷에 공개된 상태입니다! Netlify 제공 URL로 바로 접속 가능합니다.

Step 2. 도메인 구매 (uniworldtest.co)

커스텀 도메인을 연결하려면 먼저 도메인을 구매해야 합니다.

1

도메인 등록 업체 선택

업체.co 도메인 가격 (연간)추천도
Namecheap약 $25~30⭐⭐⭐⭐⭐
Cloudflare Registrar약 $10~15 (원가)⭐⭐⭐⭐⭐
가비아 (한국)약 ₩30,000~40,000⭐⭐⭐⭐
Google Domains약 $30⭐⭐⭐⭐
💡 추천: Namecheap이나 Cloudflare가 가격이 저렴하고 DNS 설정이 편합니다. 한국어 지원이 필요하면 가비아를 추천합니다.
2

도메인 구매 과정

1. 선택한 업체에서 uniworldtest.co를 검색합니다.

2. 사용 가능하면 장바구니에 담고 결제합니다.

3. WHOIS 보호(Privacy Protection)를 꼭 활성화하세요 (개인정보 보호).

Step 3. Netlify에 커스텀 도메인 연결

1

Netlify에서 도메인 추가

Netlify 대시보드에서:

# 1. 배포된 사이트 클릭
# 2. Domain management(도메인 관리) → Add custom domain 클릭
# 3. uniworldtest.co 입력 → Verify → Add domain
2

DNS 설정 (도메인 등록 업체에서)

도메인을 구매한 업체의 DNS 관리 페이지에서 다음 레코드를 추가합니다:

방법 A: Netlify DNS 사용 (추천 - 가장 간단)

# Netlify가 제공하는 네임서버로 변경
# Domain management → Netlify DNS 선택 후
# 도메인 등록 업체에서 네임서버를 아래로 변경:

dns1.p01.nsone.net
dns2.p01.nsone.net
dns3.p01.nsone.net
dns4.p01.nsone.net

방법 B: 기존 DNS에 레코드 추가

# A 레코드 (루트 도메인)
Type: A
Host: @
Value: 75.2.60.5

# CNAME 레코드 (www 서브도메인)
Type: CNAME
Host: www
Value: [your-site-name].netlify.app
⚠️ 주의: DNS 변경이 전파되는 데 최대 24~48시간이 걸릴 수 있습니다. 보통은 10분~1시간 내에 작동합니다.
3

HTTPS(SSL) 인증서 활성화

DNS가 연결되면 Netlify가 자동으로 무료 SSL 인증서(Let's Encrypt)를 발급합니다.

# Netlify 대시보드에서:
# Domain management → HTTPS → Verify DNS configuration
# → Provision certificate 클릭
✅ 완료! 이제 https://uniworldtest.co로 접속하면 웹사이트가 표시됩니다!
